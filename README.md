# 诗境雅集 - AI诗词创作平台

一个集诗词欣赏、AI创作、社交互动于一体的现代化诗词文化平台。

## ✨ 功能特色

### 🏠 诗词欣赏
- **精选诗词库**：收录经典诗词作品，支持按朝代、作者、体裁筛选
- **智能搜索**：支持按标题、作者、内容、主题等多维度搜索
- **沉浸式体验**：优美的界面设计，营造诗意氛围
- **名句轮播**：精选名句自动播放，支持手动控制

### ✨ AI创作工坊
- **多种创作模式**：
  - AI辅助创作：智能生成诗词建议
  - 模板填词：基于经典格律模板
  - 灵感激发：创意思维启发
  - 接龙创作：多人协作完成

- **智能创作助手**：
  - 主题灵感推荐
  - 韵律检查分析
  - 押韵词典查询
  - 对仗建议

- **支持多种体裁**：
  - 五言绝句/律诗
  - 七言绝句/律诗
  - 词
  - 古风

### 📚 作品管理
- **个人作品库**：统一管理所有创作作品
- **作品统计**：点赞数、收藏数、浏览数等数据分析
- **创作历程**：记录创作成长轨迹
- **标签云**：可视化展示常用创作主题
- **导出功能**：支持作品导出为文本文件

### 🌸 诗友社区
- **作品发布**：分享原创作品到社区
- **互动交流**：点赞、评论、收藏、分享
- **热门话题**：发现当前热门创作主题
- **推荐诗友**：发现志同道合的诗词爱好者
- **创作挑战**：参与社区创作活动
- **社区公告**：获取最新平台动态

## 🚀 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **数据库**：Supabase (PostgreSQL)
- **认证系统**：Supabase Auth
- **文件存储**：Supabase Storage
- **AI服务**：自定义AI诗词生成服务

## 📦 安装与运行

### 环境要求
- Node.js >= 20.19.0
- npm 或 yarn

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd poetry-platform
```

2. **安装依赖**
```bash
npm install
```

3. **环境配置**
创建 `.env.local` 文件并配置以下环境变量：
```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_FUNCTION_URL=your_supabase_functions_url
```

4. **数据库设置**
在 Supabase 控制台的 SQL 编辑器中执行 `supabase-schema.sql` 文件中的所有 SQL 语句来创建数据库表结构。

5. **启动开发服务器**
```bash
npm run dev
```

6. **构建生产版本**
```bash
npm run build
```

## 🔧 核心功能说明

### AI创作流程
1. 选择创作模式（AI辅助、模板填词等）
2. 输入创作主题和情感基调
3. 选择诗词体裁
4. 添加关键意象（可选）
5. 点击"开始创作"生成作品
6. 编辑、保存或分享生成的作品

### 韵律检查
- 自动分析诗词的平仄格律
- 检查押韵是否符合规范
- 提供优化建议

### 押韵词典
- 输入任意汉字查找押韵字
- 显示声调信息
- 支持多种韵部

## 🎯 使用技巧

### 创作建议
1. **主题选择**：选择具体而有画面感的主题，如"雨后初晴的山林"
2. **情感表达**：明确要表达的情感，有助于AI生成更贴切的作品
3. **体裁选择**：初学者建议从七言绝句开始
4. **关键词使用**：添加2-3个核心意象词，如"明月、青山、流水"

### 平台使用
1. **收藏功能**：遇到喜欢的诗词及时收藏
2. **笔记记录**：在诗词页面记录自己的理解和感悟
3. **社区互动**：积极参与评论和讨论，提升创作水平
4. **定期练习**：建议每日创作，培养诗词语感
